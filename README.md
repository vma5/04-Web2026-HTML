# 04-Web2026-HTML


🍳 Sabores do Mundo - Site de Receitas
Este é um projeto de site de receitas com três páginas, desenvolvido em HTML5 e CSS3. O objetivo é oferecer uma interface limpa, visualmente atraente e responsiva para exibir pratos e instruções culinárias.

🚀 Funcionalidades
Página Inicial: Destaque para a receita do dia e boas-vindas.

Página de Categorias: Navegação organizada por tipos de pratos (Massas, Sobremesas, Saudáveis).

Página de Detalhes: Visualização completa da receita, incluindo lista de ingredientes, modo de preparo e suporte a vídeo explicativo.

Design Responsivo: O layout se ajusta a diferentes tamanhos de tela.

📁 Estrutura do Projeto
Plaintext
/
├── index.html            # Página inicial do site
├── categorias.html       # Lista de categorias de comida
├── receita-detalhe.html  # Página detalhada da Lasanha de Manjericão
├── image_0.png           # Banner dp Menu
├── image_1.png           # Foto do prato principal
├── image_2.png           # Foto dos ingredientes
└── image_3.png           # Mosaico das categorias


🛠️ Tecnologias Utilizadas
HTML: Estrutura semântica do conteúdo.

YouTube Embed: Integração de vídeo externo para o passo a passo.

⚙️ Como usar
Baixe todos os arquivos do repositório para uma pasta em seu computador.

Certifique-se de que as imagens (image_0.png até image_3.png) estejam na mesma pasta que os arquivos HTML.

Abra o arquivo index.html em qualquer navegador (Chrome, Firefox, Edge.).

Navegue entre as páginas usando o menu superior.

📝 Notas sobre o Vídeo
O vídeo na página de detalhes utiliza a tag <iframe>. Para alterar o vídeo, basta substituir o ID no link https://www.youtube.com/embed/ID_DO_VIDEO dentro do código HTML.
